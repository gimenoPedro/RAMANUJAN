This repository contains the three versions of the RAMANUJAN code. RAMANUJAN is the undergraduate thesis project (in Portuguese, *Trabalho de Conclusão de Curso*) developed by Pedro Gimeno, an Electrical Engineering student at the University of Campinas (UNICAMP), Brazil.

The purpose of this project was to develop an intelligent calculator capable of solving elementary-level physics problems through a recursive algorithm based on a graph network. In addition to the software, which is intended to be used with the Arduino IDE, the complete system includes two 16×2 LCD displays, a 4×4 matrix keypad, an ESP32-S3 microcontroller, and several LEDs used as visual indicators.

The project evolved through multiple hardware and software iterations, ultimately resulting in an embedded system capable of interactively solving physics exercises and providing feedback to the user through its display and visual signaling interface.
